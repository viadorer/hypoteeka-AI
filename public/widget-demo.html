Hugo AI Widget

Embeddable chat widget pro jakoukoliv webovou stranku. Kolecko v rohu, po kliknuti se otevre dialog s Hugem.

Instalace

Pridejte tento kod pred uzaviraci </body> tag:

<script>
  window.HugoConfig = {
    apiUrl: 'https://vas-domena.cz',
    widgetKey: 'vas-widget-key',
    primaryColor: '#E91E63',
    position: 'bottom-right',
    title: 'Hugo',
    subtitle: 'Hypotecni poradce',
    greeting: 'Ahoj! Jsem Hugo, vas hypotecni poradce. Jak vam mohu pomoci?',
  };
</script>
<script src="https://vas-domena.cz/hugo-widget.js" defer></script>

Konfigurace

ParametrTypPopis
apiUrlstringURL vaseho serveru (bez lomitka na konci)
widgetKeystringAPI klic pro autentizaci widgetu
primaryColorstringHlavni barva widgetu (hex)
positionstringbottom-right, bottom-left, top-right, top-left
titlestringJmeno v hlavicce chatu
subtitlestringPodtitulek v hlavicce
greetingstringUvodni zprava od Huga
logoUrlstringURL loga (volitelne)
zIndexnumberCSS z-index (default: 9999)
tenantIdstringID tenanta (pro multi-tenant)

JavaScript API

// Otevrit chat
HugoWidget.open();

// Zavrit chat
HugoWidget.close();

// Toggle
HugoWidget.toggle();

// Poslat zpravu programaticky
HugoWidget.sendMessage('Chci ocenit nemovitost');
